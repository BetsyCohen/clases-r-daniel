# 📊 Clases de R – Procesamiento de datos

Repositorio con materiales para aprender a procesar datos de encuestas en R, usando `tidyverse` y herramientas básicas de automatización.

---

## 📁 Estructura del proyecto

- `00_procesamiento.R` → script principal de la clase  
- `input/` → bases de datos y archivos necesarios  
- `old/` → versiones anteriores o archivos en desuso  
- `PDC.xlsx` → planificación de tablas y cruces  
- `.gitignore` → archivos que Git ignora  
- `clases-r-daniel.Rproj` → proyecto de RStudio  

---

## 🚀 Cómo usar este repo

### 1. Clonar el repositorio

Abrí la terminal (o RStudio → Terminal) y corré:

```bash
git clone https://github.com/BetsyCohen/clases-r-daniel.git
cd clases-r-daniel
```

2. Abrir el proyecto podes empezar a trabajar en el script
00_procesamiento.R

🔁 Flujo básico con Git

Cada vez que hagas cambios:


```bash
git status          # ver cambios
git add .           # agregar cambios
git commit -m "mensaje claro"
git push            # subir a GitHub
```

⚠️ Cosas a tener en cuenta
Trabajá siempre dentro del proyecto (.Rproj)
No modifiques archivos dentro de input/ (son datos originales)
Usá nombres claros para commits (no “cambios varios” 😄)

Si algo no funciona git status
